# Эксперимент: подготовка данных ордербука

## Назначение
Построение массива состояний стакана из L2-ленты и сохранение выборки в `.npy`.

## Входные данные
- `data/BTC-USDT-L2orderbook-400lv-2025-08-28.parquet`

## Скрипты
- `reconstruct.py` — реконструкция состояний и выборка.

## Выходные артефакты
- `data/orderbook_data.npy` — итоговый набор состояний стакана (формат `(N, 2, 400, 2)`).

## Примечания
- Параметры выборки задаются в `reconstruct.py`.
